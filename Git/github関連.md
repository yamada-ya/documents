# Organization
## 概要
Organization とは github の共有アカウント.
複数プロジェクト管理でき,人数も無制限.
参加者ごとに権限を設定できる.
Organization は public/private 両方の Repository を無制限に所有可能.
機能追加はアップグレードが必要.
Team という機能では Organization の中にグループを作ることができ,権限の一括管理も可能.
